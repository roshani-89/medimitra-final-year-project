# MediMitra - Discounted Medicine & Health Support Web App

## Project Overview
MediMitra is a socially impactful web application aimed at reducing medicine wastage, improving access to medicines for needy people, and promoting health awareness. It provides a platform for patients, donors, and NGOs to connect and support each other.

## Modules
1. User Registration & Login
2. Product Management (Medicines & Medical Equipment)
3. Buyer Management
4. Location-Based Matching
5. AI Health Assistant (Chatbot)
6. Health Tips & Awareness
7. Emergency Contact

## Technologies Used
- Frontend: ReactJS, TailwindCSS
- Backend: Node.js, Express.js, MongoDB

## Setup Instructions
- Backend and frontend will be in separate folders.
- Use `npm install` in each folder to install dependencies.
- Run backend server with `npm start`.
- Run frontend with `npm start`.
